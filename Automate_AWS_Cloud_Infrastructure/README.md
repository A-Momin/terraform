<details><summary style="font-size:25px;color:Orange;text-align:left">How to initialize and run  the project</summary>

1. Init the Terraform project
    - `$ cd .`
    - `$ terraform init`
2. Check the execution plan
    - `$ terraform plan`
3. Apply the execution plan
    - `$ terraform apply`
4. Destroy all the resources:
    - `$ terraform destroy`

</details>
